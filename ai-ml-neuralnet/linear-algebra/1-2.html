Advanced Vector Spaces - Theory

A Comprehensive Exploration of Abstract Vector Spaces, Linear Transformations, and Inner Product Spaces

1.2.1 Vector Spaces - Definition and Axioms

auto_stories Definition

A vector space is a set V equipped with two operations:

• Vector addition: + : V × V → V
• Scalar multiplication: · : ℝ × V → V

satisfying the following 10 axioms for all u, v, w ∈ V and a, b ∈ ℝ

add_circle Addition Axioms

  • Closure: u + v ∈ V
  • Commutativity: u + v = v + u
  • Associativity: (u + v) + w = u + (v + w)
  • Zero Vector: ∃ 0 ∈ V such that u + 0 = u
  • Additive Inverse: ∃ -u ∈ V such that u + (-u) = 0

close Scalar Multiplication Axioms

  • Closure: a·u ∈ V
  • Distributivity (vector): a·(u + v) = a·u + a·v
  • Distributivity (scalar): (a + b)·u = a·u + b·u
  • Associativity: (ab)·u = a·(b·u)
  • Identity: 1·u = u

1.2.2 Examples of Vector Spaces

Vector spaces appear throughout mathematics and its applications. Here are some fundamental examples:

grid_on Euclidean Space

ℝⁿ: n-tuples of real numbers

• Addition: (x₁,...,xₙ) + (y₁,...,yₙ) = (x₁+y₁,...,xₙ+yₙ)

• Scalar mult: c(x₁,...,xₙ) = (cx₁,...,cxₙ)

• Dimension: n

• Standard basis: e₁=(1,0,...,0), e₂=(0,1,...,0), ..., eₙ=(0,0,...,1)

Most intuitive example with clear geometric interpretation

functions Function Spaces

F[a,b]: Real-valued functions on interval [a,b]

• Addition: (f+g)(x) = f(x) + g(x)

• Scalar mult: (cf)(x) = c·f(x)

• Zero vector: 0(x) = 0 for all x

• Subspaces: C[a,b] (continuous), Cⁿ[a,b] (n-times differentiable)

Infinite-dimensional space with applications in differential equations

view_column Matrix Spaces

M_{m×n}: m×n matrices with real entries

• Addition: (A+B)_{ij} = A_{ij} + B_{ij}

• Scalar mult: (cA)_{ij} = c·A_{ij}

• Zero vector: 0_{m×n} (all entries zero)

• Dimension: m×n

• Basis: matrices with single 1 and rest 0

Fundamental in linear transformations and data representation

show_chart Polynomial Spaces

Pₙ: Polynomials of degree ≤ n

• Addition: (p+q)(x) = p(x) + q(x)

• Scalar mult: (cp)(x) = c·p(x)

• Zero vector: p(x) = 0 (zero polynomial)

• Dimension: n+1

• Standard basis: {1, x, x², ..., xⁿ}

Used in approximation theory and interpolation

1.2.3 Subspaces - Definition and Properties

auto_stories Definition

A subspace of a vector space V is a subset W ⊆ V that is itself a vector space under the operations inherited from V.

rule Subspace Criterion

  • Non-empty: W ≠ ∅
  • Closed under addition: ∀ u,v ∈ W: u+v ∈ W
  • Closed under scalar multiplication: ∀ v ∈ W, ∀ c ∈ ℝ: c·v ∈ W

lightbulb Simplified Test

W is a subspace of V if and only if:

∀ u,v ∈ W, ∀ c ∈ ℝ: u+v ∈ W and c·u ∈ W

category Key Properties

  • Contains zero vector: 0 ∈ W
  • Closed under linear combinations: ∀ v₁,...,vₙ ∈ W, ∀ c₁,...,cₙ ∈ ℝ: c₁v₁ + ... + cₙvₙ ∈ W
  • Inherits operations: Addition and scalar multiplication in W are the same as in V
  • Intersection property: The intersection of any collection of subspaces is a subspace
  • Dimension constraint: dim(W) ≤ dim(V)

1.2.4 Subspace Examples and Tests

check_circle Valid Subspace Examples

Lines through origin in ℝ²

• Set: {(x,y) | y = mx} for fixed m

• Contains: (0,0)

• Closed under: addition and scalar multiplication


Solution sets of homogeneous systems

• Set: {x | Ax = 0} for matrix A

• Called: null space of A

• Dimension: n - rank(A)

Verification Process

1. Check if 0 ∈ W
2. Verify u,v ∈ W ⇒ u+v ∈ W
3. Verify v ∈ W, c ∈ ℝ ⇒ c·v ∈ W

cancel Non-Subspace Examples

Lines not through origin in ℝ²

• Set: {(x,y) | y = mx + b} where b ≠ 0

• Fails: Does not contain (0,0)

• Also: Not closed under addition or scalar multiplication


Unit circle in ℝ²

• Set: {(x,y) | x² + y² = 1}

• Contains: (1,0) and (0,1)

• Fails: (1,0) + (0,1) = (1,1) not on unit circle

• Also: 2·(1,0) = (2,0) not on unit circle

Common Pitfalls

1. Sets not containing zero vector
2. Sets bounded in some way
3. Sets defined by non-linear equations
4. Sets with constraints on vector magnitude

functions Function Subspaces

Even functions in F[-a,a]

• Set: {f | f(-x) = f(x) for all x}

• Contains: zero function f(x) = 0

• Closed under: addition and scalar multiplication


Polynomials of degree ≤ n in F[a,b]

• Set: {p | p(x) = a₀ + a₁x + ... + aₙxⁿ}

• Dimension: n+1

• Basis: {1, x, x², ..., xⁿ}

Function Space Tests

1. Verify zero function satisfies the condition
2. Check if property preserved under addition
3. Check if property preserved under scalar multiplication

view_column Matrix Subspaces

Symmetric matrices in M_{n×n}

• Set: {A | Aᵀ = A}

• Contains: zero matrix

• Dimension: n(n+1)/2


Upper triangular matrices in M_{n×n}

• Set: {A | A_{ij} = 0 for i > j}

• Contains: zero matrix

• Dimension: n(n+1)/2

Matrix Space Tests

1. Verify zero matrix satisfies the condition
2. Check if property preserved under addition
3. Check if property preserved under scalar multiplication

1.2.5 Linear Independence and Dependence

auto_stories Linear Independence

A set of vectors {v₁, v₂, ..., vₙ} is linearly independent if the only solution to:

c₁v₁ + c₂v₂ + ... + cₙvₙ = 0

is c₁ = c₂ = ... = cₙ = 0

auto_stories Linear Dependence

A set of vectors {v₁, v₂, ..., vₙ} is linearly dependent if there exist scalars c₁, c₂, ..., cₙ, not all zero, such that:

c₁v₁ + c₂v₂ + ... + cₙvₙ = 0

rule Key Properties

  • Unique representation: In a linearly independent set, each vector has a unique representation as a linear combination
  • Redundancy: In a linearly dependent set, at least one vector is a linear combination of the others
  • Zero vector: Any set containing the zero vector is linearly dependent
  • Subset property: Every subset of a linearly independent set is linearly independent

Test Method

Form matrix A with vectors as columns and solve Ax = 0. If only trivial solution exists, vectors are linearly independent.

psychology Geometric Interpretation

  • In ℝ²: Two vectors are linearly independent if they point in different directions (not parallel)
  • In ℝ³: Three vectors are linearly independent if they don't lie in the same plane
  • General case: Vectors are linearly independent if each adds a new dimension to the space they span
  • Maximum size: In an n-dimensional space, any set with more than n vectors is linearly dependent

Visualization

Linear independence means no vector is "redundant" - each provides unique information that cannot be replicated by the others.

1.2.6 Basis and Dimension

auto_stories Basis

A basis for a vector space V is a linearly independent set of vectors that spans V.

• Every vector in V can be uniquely expressed as a linear combination of basis vectors
• Minimal spanning set
• Maximal linearly independent set

straighten Dimension

The dimension of a vector space V, denoted dim(V), is the number of vectors in any basis for V.

• All bases for a vector space have the same number of elements
• dim(ℝⁿ) = n
• dim(Pₙ) = n+1 (polynomials of degree ≤ n)

rule Key Properties

  • Unique representation: For any basis B = {v₁,...,vₙ}, every v ∈ V has a unique representation v = c₁v₁ + ... + cₙvₙ
  • Extension theorem: Any linearly independent set can be extended to a basis
  • Reduction theorem: Any spanning set can be reduced to a basis
  • Subspace dimension: If W ⊆ V, then dim(W) ≤ dim(V)

lightbulb Basis Theorem

If V is finite-dimensional with dim(V) = n, then:
• Any linearly independent set with n vectors is a basis
• Any spanning set with n vectors is a basis

category Standard Bases

  • ℝⁿ: Standard basis {e₁,e₂,...,eₙ} where eᵢ has 1 in position i and 0 elsewhere
  • Pₙ: Standard basis {1,x,x²,...,xⁿ}
  • M_{m×n}: Standard basis consists of matrices with single 1 and rest 0
  • Function spaces: Various bases including orthogonal polynomials, Fourier basis
Example:
Standard basis for ℝ³: e₁ = (1,0,0), e₂ = (0,1,0), e₃ = (0,0,1)
Any vector v = (a,b,c) can be written as v = ae₁ + be₂ + ce₃

1.2.7 Coordinate Systems and Change of Basis

grid_on Coordinate Systems

Given a basis B = {v₁, v₂, ..., vₙ} for vector space V, any vector v ∈ V can be uniquely represented as:

v = c₁v₁ + c₂v₂ + ... + cₙvₙ

The coordinate vector of v relative to B is:

[v]₈ = [c₁, c₂, ..., cₙ]ᵀ
Example:
In ℝ² with standard basis B = {e₁, e₂}, the vector v = (3, 2) has coordinates [v]₈ = [3, 2]ᵀ

transform Change of Basis Matrix

Let B = {v₁, v₂, ..., vₙ} and C = {w₁, w₂, ..., wₙ} be two bases for V.

The change of basis matrix from B to C, denoted P_{C←B}, satisfies:

[v]C = P_{C←B} [v]B
  • Columns of P_{C←B} are [v₁]C, [v₂]C, ..., [vₙ]C
  • P_{C←B} is always invertible
  • P_{B←C} = (P_{C←B})⁻¹

swap_horiz Computing Change of Basis

To find P_{C←B}:

  • Express each basis vector of B in terms of C
  • For each vᵢ ∈ B, find [vᵢ]C
  • These coordinate vectors form the columns of P_{C←B}
Example:
In ℝ², if B = {e₁, e₂} and C = {(1,1), (1,-1)}, then:
P_{C←B} = [[1/2, 1/2], [1/2, -1/2]]

insights Applications and Properties

  • Linear transformations: Matrix representation depends on basis choice
  • Similar matrices: A and B similar if B = P⁻¹AP for some invertible P
  • Diagonalization: Finding basis where transformation matrix is diagonal
  • Computational efficiency: Some problems simpler in specific bases
[T(v)]C = [T]_{C←B} [v]B
[T]_{C←C} = P_{C←B} [T]_{B←B} P_{B←C}

1.2.8 Linear Transformations - Definition and Properties

auto_stories Definition

A linear transformation is a function T: V → W between vector spaces that satisfies:

• Additivity: T(u + v) = T(u) + T(v) for all u, v ∈ V
• Homogeneity: T(c·v) = c·T(v) for all v ∈ V and all scalars c

Equivalently, T preserves linear combinations:
T(c₁v₁ + c₂v₂ + ... + cₙvₙ) = c₁T(v₁) + c₂T(v₂) + ... + cₙT(vₙ)

rule Key Properties

  • Zero vector preservation: T(0_V) = 0_W
  • Inverse preservation: T(-v) = -T(v)
  • Subspace mapping: Image of subspace is subspace
  • Linear independence: Maps independent sets to independent sets if injective
  • Composition: Composition of linear transformations is linear
Example:
The derivative operator D: Pₙ → Pₙ₋₁ defined by D(p) = p' is a linear transformation.

category Types of Linear Transformations

  • Isomorphism: Bijective linear transformation
  • Endomorphism: T: V → V (maps space to itself)
  • Automorphism: Bijective endomorphism
  • Projection: T∘T = T (idempotent)
  • Nilpotent: Tᵏ = 0 for some k
Example:
The rotation matrix R_θ = [cosθ -sinθ; sinθ cosθ] is an automorphism of ℝ².

calculate Operations on Linear Transformations

  • Addition: (T + S)(v) = T(v) + S(v)
  • Scalar multiplication: (cT)(v) = c·T(v)
  • Composition: (T∘S)(v) = T(S(v))
  • Inverse: T⁻¹(T(v)) = v (if bijective)
  • Adjoint: ⟨T(u),v⟩ = ⟨u,T*(v)⟩ (in inner product spaces)

psychology Geometric Interpretation

  • Preserves lines: Maps lines to lines
  • Preserves origin: Maps origin to origin
  • Preserves parallelism: Parallel lines map to parallel lines
  • Uniform scaling: Scales all vectors in a subspace uniformly
  • Structure preservation: Maintains vector space structure

1.2.9 Matrix Representation of Linear Transformations

grid_on Standard Matrix Representation

For a linear transformation T: ℝⁿ → ℝᵐ, the standard matrix [T] is constructed by:

  • Apply T to standard basis vectors e₁, e₂, ..., eₙ
  • Results T(e₁), T(e₂), ..., T(eₙ) form columns of [T]
  • For any v ∈ ℝⁿ: T(v) = [T]v
[T] = [T(e₁) | T(e₂) | ... | T(eₙ)]
Example:
For rotation T: ℝ² → ℝ² by angle θ:
T(e₁) = (cosθ, sinθ), T(e₂) = (-sinθ, cosθ)
[T] = [cosθ -sinθ; sinθ cosθ]

swap_horiz General Matrix Representation

For T: V → W with bases B = {v₁,...,vₙ} for V and C = {w₁,...,wₘ} for W:

  • Apply T to each basis vector vⱼ
  • Express T(vⱼ) in basis C: T(vⱼ) = Σᵢ aᵢⱼwᵢ
  • Matrix [T]_{C←B} has entries aᵢⱼ
  • For any v ∈ V: [T(v)]C = [T]_{C←B} [v]B
[T]_{C←B} = [[T(v₁)]C | [T(v₂)]C | ... | [T(vₙ)]C]

transform Change of Basis Effects

Matrix representation depends on choice of bases:

  • If B', C' are new bases:
  • [T]_{C'←B'} = P_{C'←C} [T]_{C←B} P_{B←B'}
  • Where P are change of basis matrices
  • For T: V → V with same basis: [T]_{B'} = P⁻¹[T]_B P
Similarity:
Matrices representing the same transformation with different bases are similar

calculate Operations and Properties

  • Composition: [T∘S]_{C←A} = [T]_{C←B} [S]_{B←A}
  • Inverse: If T invertible, [T⁻¹]_{B←C} = ([T]_{C←B})⁻¹
  • Rank: rank(T) = rank([T])
  • Nullity: nullity(T) = nullity([T])
  • Eigenvalues: Same for all matrix representations of T
dim(V) = rank(T) + nullity(T)

1.2.10 Kernel and Range of Linear Transformations

call_split Kernel (Null Space)

For a linear transformation T: V → W, the kernel is:

ker(T) = {v ∈ V | T(v) = 0}

• Subspace of domain V
• Measures how much T "collapses" the space
• Also called null space of T

call_merge Range (Image)

For a linear transformation T: V → W, the range is:

range(T) = {T(v) | v ∈ V}

• Subspace of codomain W
• All possible outputs of T
• Also called image of T

rule Key Properties

  • Subspace property: Both ker(T) and range(T) are subspaces
  • Injectivity: T is injective iff ker(T) = {0}
  • Surjectivity: T is surjective iff range(T) = W
  • Composition: ker(T) ⊆ ker(S∘T) and range(S∘T) ⊆ range(S)
  • Matrix connection: For matrix A, ker(A) = null(A) and range(A) = col(A)
Example:
For T: ℝ³ → ℝ² defined by T(x,y,z) = (x,y):
• ker(T) = {(0,0,z) | z ∈ ℝ}
• range(T) = ℝ²

calculate Computational Methods

  • Kernel calculation: Solve T(v) = 0 or Ax = 0
  • Range calculation: Span of columns of matrix or {T(v₁),...,T(vₙ)} for basis {v₁,...,vₙ}
  • Dimension: nullity(T) = dim(ker(T)), rank(T) = dim(range(T))
  • Basis finding: Use Gaussian elimination for matrix representations

lightbulb Rank-Nullity Theorem

For any linear transformation T: V → W with dim(V) = n:

dim(ker(T)) + dim(range(T)) = n

Or equivalently: nullity(T) + rank(T) = dim(V)

psychology Geometric Interpretation

  • Kernel: "Collapsed" dimension - vectors mapped to zero
  • Range: "Output" dimension - space covered by transformation
  • Projection: For projection P, ker(P) is orthogonal to range(P)
  • Rotation: For rotation R, ker(R) = {0} and range(R) is entire space
  • Dimension reduction: dim(ker(T)) measures "lost" dimensions

insights Applications

  • Differential equations: Solution spaces as kernels of differential operators
  • Computer graphics: Projection transformations and their null spaces
  • Data compression: Range of transformation represents compressed data
  • Machine learning: Feature spaces as ranges of embedding transformations
  • Control theory: Controllable and observable subspaces

1.2.11 Rank-Nullity Theorem and Applications

lightbulb Rank-Nullity Theorem

For any linear transformation T: V → W between finite-dimensional vector spaces:

dim(ker(T)) + dim(range(T)) = dim(V)

Equivalently, for any matrix A ∈ M_{m×n}:

nullity(A) + rank(A) = n

calculate Matrix Applications

  • System analysis: Determines solution space dimension for Ax = b
  • Matrix inversion: A invertible iff nullity(A) = 0 and rank(A) = n
  • Least squares: Relates to rank of design matrix
  • SVD computation: Number of non-zero singular values equals rank
Example:
For A = [1 2 3; 4 5 6], rank(A) = 2, so nullity(A) = 3 - 2 = 1

psychology Linear Transformation Applications

  • Injectivity test: T injective iff nullity(T) = 0
  • Surjectivity test: T surjective iff rank(T) = dim(W)
  • Isomorphism: T isomorphism iff nullity(T) = 0 and rank(T) = dim(V) = dim(W)
  • Dimension analysis: Predicts dimension of solution spaces
Example:
For projection P: ℝ³ → ℝ², rank(P) = 2, so nullity(P) = 3 - 2 = 1

science Scientific Applications

  • Differential equations: Solution space dimension analysis
  • Quantum mechanics: State space and observable operators
  • Signal processing: Filter design and frequency analysis
  • Control theory: Controllability and observability

trending_up Machine Learning Applications

  • Feature spaces: Dimension of embedding spaces
  • Neural networks: Rank of weight matrices
  • PCA: Dimension reduction via rank truncation
  • Data compression: Low-rank approximations

1.2.12 Inner Product Spaces - Definition and Properties

auto_stories Definition

An inner product on a vector space V is a function ⟨·,·⟩: V×V→ℝ satisfying:

• Symmetry: ⟨u,v⟩ = ⟨v,u⟩ for all u,v ∈ V
• Linearity: ⟨au+bv,w⟩ = a⟨u,w⟩ + b⟨v,w⟩ for all u,v,w ∈ V and a,b ∈ ℝ
• Positive-definiteness: ⟨v,v⟩ ≥ 0 for all v ∈ V, and ⟨v,v⟩ = 0 iff v = 0

A vector space equipped with an inner product is called an inner product space.

straighten Norm and Distance

  • Norm: ||v|| = √⟨v,v⟩ (length of vector)
  • Distance: d(u,v) = ||u-v|| (distance between vectors)
  • Unit vector: û = v/||v|| (normalized vector)
  • Cauchy-Schwarz: |⟨u,v⟩| ≤ ||u||·||v||
||v|| ≥ 0 and ||v|| = 0 iff v = 0
||cv|| = |c|·||v|| for any scalar c

category Examples of Inner Products

  • Dot product in ℝⁿ: ⟨u,v⟩ = u₁v₁ + u₂v₂ + ... + uₙvₙ
  • Weighted inner product: ⟨u,v⟩_w = Σwᵢuᵢvᵢ where wᵢ > 0
  • Function space: ⟨f,g⟩ = ∫f(x)g(x)dx over interval [a,b]
  • Matrix space: ⟨A,B⟩ = tr(AᵀB) (Frobenius inner product)
Example:
In ℝ² with standard inner product, ⟨(3,4),(1,2)⟩ = 3·1 + 4·2 = 11

rule Key Properties

  • Orthogonality: u ⊥ v if ⟨u,v⟩ = 0
  • Pythagorean theorem: If u ⊥ v, then ||u+v||² = ||u||² + ||v||²
  • Parallelogram law: ||u+v||² + ||u-v||² = 2(||u||² + ||v||²)
  • Polarization identity: ⟨u,v⟩ = ¼(||u+v||² - ||u-v||²)
⟨u,v⟩ = ½(||u+v||² - ||u||² - ||v||²)

psychology Geometric Interpretation

  • Angle: cos θ = ⟨u,v⟩/(||u||·||v||)
  • Projection: projᵥu = (⟨u,v⟩/⟨v,v⟩)v
  • Orthogonal complement: W⊥ = {v | ⟨v,w⟩ = 0 for all w ∈ W}
  • Orthogonal decomposition: V = W ⊕ W⊥ for any subspace W
Example:
In ℝ³, the angle between v = (1,0,0) and w = (1,1,0) is θ = 45°

1.2.13 Orthogonality and Orthogonal Sets

auto_stories Orthogonality

Two vectors u and v in an inner product space are orthogonal if their inner product is zero:

u ⊥ v if and only if ⟨u,v⟩ = 0

• Orthogonal vectors are linearly independent
• Zero vector is orthogonal to all vectors
• Generalizes the concept of perpendicularity

category Orthogonal Sets

A set S = {v₁, v₂, ..., vₙ} is an orthogonal set if all vectors in S are pairwise orthogonal:

⟨vᵢ, vⱼ⟩ = 0 for all i ≠ j

• If additionally ||vᵢ|| = 1 for all i, then S is an orthonormal set
• Orthogonal sets of non-zero vectors are linearly independent

rule Key Properties

  • Linear independence: Any orthogonal set of non-zero vectors is linearly independent
  • Pythagorean theorem: If u ⊥ v, then ||u+v||² = ||u||² + ||v||²
  • Orthogonal decomposition: Any vector can be written as sum of orthogonal components
  • Basis property: Orthogonal basis simplifies coordinate calculations
If {v₁,...,vₙ} is orthogonal, then ||c₁v₁+...+cₙvₙ||² = |c₁|²||v₁||² + ... + |cₙ|²||vₙ||²

calculate Orthogonal Complements

  • Definition: W⊥ = {v ∈ V | ⟨v,w⟩ = 0 for all w ∈ W}
  • Subspace property: W⊥ is always a subspace
  • Double complement: (W⊥)⊥ = W for finite-dimensional subspaces
  • Direct sum: V = W ⊕ W⊥
  • Dimension relation: dim(W) + dim(W⊥) = dim(V)
Example:
In ℝ³, if W = span{(1,0,0)}, then W⊥ = span{(0,1,0), (0,0,1)}

psychology Orthogonal Projections

  • Projection formula: proj_W(v) = Σ⟨v,uᵢ⟩uᵢ for orthonormal basis {uᵢ} of W
  • Orthogonal projection: v = proj_W(v) + proj_{W⊥}(v)
  • Best approximation: proj_W(v) is closest vector in W to v
  • Projection matrix: P = A(AᵀA)⁻¹Aᵀ where columns of A form basis for W
||v - proj_W(v)|| ≤ ||v - w|| for all w ∈ W

apps Applications

  • Least squares: Solving overdetermined systems
  • Fourier series: Function approximation with orthogonal basis
  • Signal processing: Orthogonal frequency division
  • Quantum mechanics: Orthogonal state vectors
  • Machine learning: Feature extraction, dimensionality reduction
Example:
QR decomposition uses orthogonal matrices for numerical stability

1.2.14 Gram-Schmidt Process and QR Decomposition

swap_vert Gram-Schmidt Process

Algorithm to convert a basis {v₁, v₂, ..., vₙ} into an orthogonal basis {u₁, u₂, ..., uₙ}:

u₁ = v₁
u₂ = v₂ - proj_{u₁}(v₂)
u₃ = v₃ - proj_{u₁}(v₃) - proj_{u₂}(v₃)
...
uₖ = vₖ - Σ_{i=1}^{k-1} proj_{uᵢ}(vₖ)

Where proj_{u}(v) = (⟨v,u⟩/⟨u,u⟩)u

uₖ = vₖ - Σ_{i=1}^{k-1} (⟨vₖ,uᵢ⟩/⟨uᵢ,uᵢ⟩)uᵢ

To get orthonormal basis, normalize each uₖ:

êₖ = uₖ/||uₖ||

grid_view QR Decomposition

Factorization of matrix A ∈ ℝᵐˣⁿ as A = QR where:

  • Q: m × m orthogonal matrix (QᵀQ = I)
  • R: m × n upper triangular matrix

For rank(A) = n, R has positive diagonal entries

A = [a₁ | a₂ | ... | aₙ] = Q [r₁₁ r₁₂ ... r₁ₙ; 0 r₂₂ ... r₂ₙ; ⋱ ⋮; 0 ... 0 rₙₙ]

Connection:

1.2.15 Orthogonal Projections and Applications

call_split Orthogonal Projection

For a subspace W of inner product space V, the orthogonal projection of vector v ∈ V onto W is the unique vector proj_W(v) ∈ W such that:

v - proj_W(v) ⊥ W
  • Orthogonal decomposition: v = proj_W(v) + proj_{W⊥}(v)
  • Projection formula: If {u₁,...,uₖ} orthonormal basis for W: proj_W(v) = Σ⟨v,uᵢ⟩uᵢ
  • Idempotent: proj_W∘proj_W = proj_W
Example:
In ℝ³, projection of v = (1,2,3) onto W = span{(1,0,0), (0,1,0)} is proj_W(v) = (1,2,0)

grid_on Projection Matrix

For subspace W with basis vectors as columns of A, the projection matrix is:

P = A(AᵀA)⁻¹Aᵀ
  • Properties: P² = P, Pᵀ = P
  • Orthogonal case: If A has orthonormal columns, P = AAᵀ
  • Projection formula: proj_W(v) = Pv
  • Residual: v - Pv = (I-P)v
Example:
For W = span{(1,1)} in ℝ², A = [1; 1], P = ½[1 1; 1 1]

calculate Key Properties

  • Best approximation: proj_W(v) is closest point in W to v
  • Minimum distance: ||v - proj_W(v)|| ≤ ||v - w|| for all w ∈ W
  • Norm inequality: ||proj_W(v)|| ≤ ||v||
  • Pythagorean theorem: ||v||² = ||proj_W(v)||² + ||proj_{W⊥}(v)||²
  • Linearity: proj_W(au+bv) = a·proj_W(u) + b·proj_W(v)
||v - proj_W(v)|| = min_{w∈W} ||v - w||

apps Applications

  • Least squares: Minimize ||Ax - b||² via x = (AᵀA)⁻¹Aᵀb
  • Signal processing: Noise filtering, signal reconstruction
  • Computer graphics: Shadow projection, 3D rendering
  • Statistics: Regression analysis, ANOVA
  • Quantum mechanics: Measurement operators
Example - Least Squares:
For overdetermined system Ax = b, the solution minimizing ||Ax - b||² is x = (AᵀA)⁻¹Aᵀb

1.2.16 Normed Spaces and Metrics

straighten Normed Spaces

A norm on a vector space V is a function ||·||: V → ℝ satisfying:

• Positive definiteness: ||v|| ≥ 0 and ||v|| = 0 iff v = 0
• Absolute homogeneity: ||αv|| = |α|·||v|| for all scalars α
• Triangle inequality: ||u+v|| ≤ ||u|| + ||v||

A vector space equipped with a norm is called a normed space.

swap_horiz Metrics

A metric (distance function) on a set X is a function d: X×X → ℝ satisfying:

• Non-negativity: d(x,y) ≥ 0 and d(x,y) = 0 iff x = y
• Symmetry: d(x,y) = d(y,x)
• Triangle inequality: d(x,z) ≤ d(x,y) + d(y,z)

Every norm induces a metric: d(u,v) = ||u-v||

category Examples of Norms

  • Euclidean norm (ℓ²): ||v||₂ = √(Σ|vᵢ|²)
  • Taxicab norm (ℓ¹): ||v||₁ = Σ|vᵢ|
  • Maximum norm (ℓ^∞): ||v||_∞ = max|vᵢ|
  • p-norm: ||v||ₚ = (Σ|vᵢ|ᵖ)^(¹/ᵖ) for p ≥ 1
Example:
For v = (3,4) in ℝ²:
• ||v||₂ = √(3²+4²) = 5
• ||v||₁ = |3|+|4| = 7
• ||v||_∞ = max(|3|,|4|) = 4

calculate Norm Properties

  • Reverse triangle inequality: |||u|| - ||v||| ≤ ||u-v||
  • Continuity: Norm is a continuous function
  • Unit sphere: {v | ||v|| = 1} is closed and bounded
  • Equivalence of norms: In finite dimensions, all norms are equivalent
c₁||v||ₐ ≤ ||v||ᵦ ≤ c₂||v||ₐ
for some constants c₁, c₂ > 0

functions Function Space Norms

  • Supremum norm: ||f||_∞ = sup{|f(x)| | x ∈ [a,b]}
  • Lᵖ norm: ||f||ₚ = (∫|f(x)|ᵖdx)^(¹/ᵖ)
  • Sobolev norm: Includes derivatives of f
  • Matrix norms: Frobenius, operator, spectral norms
Example:
For f(x) = x² on [0,1]:
• ||f||_∞ = 1
• ||f||₂ = √(∫₀¹x⁴dx) = √(1/5) ≈ 0.447

psychology Applications

  • Optimization: Constrained minimization, gradient descent
  • Approximation theory: Best approximation in different norms
  • Machine learning: Regularization (L₁, L₂ norms)
  • Functional analysis: Banach spaces, Hilbert spaces
  • Signal processing: Signal reconstruction, compression
LASSO: minₓ ||Ax-b||₂² + λ||x||₁
Ridge: minₓ ||Ax-b||₂² + λ||x||₂²

1.2.17 Isomorphisms and Isomorphic Spaces

swap_horiz Isomorphism Definition

A linear transformation T: V → W is an isomorphism if:

  • Bijective: Both injective and surjective
  • Invertible: Exists T⁻¹: W → V such that T⁻¹∘T = id_V and T∘T⁻¹ = id_W

Two vector spaces V and W are isomorphic (V ≅ W) if there exists an isomorphism between them.

T is isomorphic iff ker(T) = {0} and range(T) = W
Example:
The map T: ℝ² → P₁ defined by T(a,b) = a + bx is an isomorphism

rule Key Properties

  • Dimension preservation: If V ≅ W, then dim(V) = dim(W)
  • Converse: If dim(V) = dim(W) = n, then V ≅ W
  • Composition: Composition of isomorphisms is an isomorphism
  • Inverse: Inverse of an isomorphism is an isomorphism
  • Equivalence relation: Isomorphism is reflexive, symmetric, and transitive
All n-dimensional vector spaces over the same field are isomorphic

calculate Constructing Isomorphisms

To construct an isomorphism T: V → W:

  • Choose bases B = {v₁,...,vₙ} for V and C = {w₁,...,wₙ} for W
  • Define T(vᵢ) = wᵢ for each basis vector
  • Extend linearly: T(Σcᵢvᵢ) = Σcᵢwᵢ

The matrix representation of T with respect to B and C is the identity matrix.

Example:
For V = P₂ and W = ℝ³, define T(a+bx+cx²) = (a,b,c)

psychology Applications and Significance

  • Classification: Classifies vector spaces up to isomorphism by dimension
  • Coordinate systems: Isomorphism between abstract space and ℝⁿ
  • Linear transformations: Study in one space applies to all isomorphic spaces
  • Duality: Isomorphism between space and its dual
  • Machine learning: Feature space isomorphisms for dimensionality reduction
V ≅ ℝⁿ where n = dim(V)

1.2.18 Dual Spaces and Applications in Machine Learning

swap_horiz Dual Space Definition

The dual space of a vector space V, denoted V*, is the space of all linear functionals on V:

V* = {f: V → ℝ | f is linear}
  • Operations: (f+g)(v) = f(v) + g(v), (cf)(v) = c·f(v)
  • Dimension: dim(V*) = dim(V) for finite-dimensional V
  • Dual basis: If {v₁,...,vₙ} is basis for V, then dual basis {f₁,...,fₙ} satisfies fᵢ(vⱼ) = δᵢⱼ
Example:
For V = ℝ³, the dual basis of standard basis is the set of projection functionals f₁(x,y,z) = x, f₂(x,y,z) = y, f₃(x,y,z) = z

transform Double Dual and Natural Isomorphism

The double dual V** is the dual of the dual space V*:

  • Natural isomorphism: φ: V → V** defined by φ(v)(f) = f(v)
  • Canonical: Does not depend on choice of basis
  • Reflexivity: V is reflexive if φ is an isomorphism
  • Finite-dimensional case: All finite-dimensional spaces are reflexive
φ(v)(f) = f(v)
Significance:
Natural isomorphism allows identifying V with V** without choosing a basis

psychology Applications in Machine Learning

  • Kernel methods: Feature maps and kernel functions as elements of dual space
  • Support Vector Machines: Dual formulation maximizes margin in dual space
  • Representation learning: Learning feature representations in dual space
  • Neural networks: Dual space interpretation of activation functions
K(x,y) = φ(x)ᵀφ(y)
where φ: X → V maps to feature space
SVM Dual Problem:
Maximize Σαᵢ - ½ΣΣαᵢαⱼyᵢyⱼK(xᵢ,xⱼ) subject to constraints

insights Advanced Concepts

  • Annihilator: W⁰ = {f ∈ V* | f(w) = 0 for all w ∈ W}
  • Transpose: For T: V → W, T*: W* → V* defined by T*(f) = f∘T
  • Adjoint: In inner product spaces, relates to transpose via Riesz representation
  • Tensors: Multilinear maps as elements of tensor product spaces
dim(W) + dim(W⁰) = dim(V)
