1. CSS Basics & Syntax

What is CSS?

CSS (Cascading Style Sheets) is a stylesheet language used to describe the presentation of a document written in HTML or XML. CSS describes how elements should be rendered on screen, on paper, in speech, or on other media.

CSS separates the presentation from the content, making it easier to maintain and update websites.

CSS Syntax

CSS syntax consists of a selector and a declaration block:

selector { property: value; property: value; }

Example:

h1 { color: blue; font-size: 24px; text-align: center; }

Ways to Include CSS

1. Inline CSS

<h1 style="color: blue; font-size: 24px;">Hello World</h1>

2. Internal CSS

<head> <style> h1 { color: blue; font-size: 24px; } </style> </head>

3. External CSS (Recommended)

<head> <link rel="stylesheet" href="styles.css"> </head>
External CSS is the best practice for most websites as it separates content from presentation and allows for better caching and maintenance.

CSS Comments

/* This is a CSS comment */ h1 { color: blue; /* This makes the text blue */ } /* Multi-line comment This can span multiple lines */

CSS Units

Absolute Units:

  • px - Pixels (1px = 1/96th of 1in)
  • pt - Points (1pt = 1/72 of 1in)
  • pc - Picas (1pc = 12pt)
  • in - Inches (1in = 96px = 2.54cm)
  • cm - Centimeters
  • mm - Millimeters

Relative Units:

  • em - Relative to the font-size of the element
  • rem - Relative to the font-size of the root element
  • % - Percentage relative to the parent element
  • vw - Viewport width (1vw = 1% of viewport width)
  • vh - Viewport height (1vh = 1% of viewport height)
  • vmin - 1% of the smaller viewport dimension
  • vmax - 1% of the larger viewport dimension
Use relative units (rem, em, %) for better responsive design and accessibility.

CSS Colors

Color Formats:

.element { color: red; /* Named color */ color: #ff0000; /* Hexadecimal */ color: #f00; /* Short hexadecimal */ color: rgb(255, 0, 0); /* RGB */ color: rgba(255, 0, 0, 0.5); /* RGBA with alpha */ color: hsl(0, 100%, 50%); /* HSL */ color: hsla(0, 100%, 50%, 0.5); /* HSLA with alpha */ }

2. CSS Selectors

Basic Selectors

Universal Selector (*)

* { margin: 0; padding: 0; box-sizing: border-box; }

Type Selector (Element Selector)

p { font-size: 16px; line-height: 1.5; }

Class Selector (.)

.container { max-width: 1200px; margin: 0 auto; }

ID Selector (#)

#header { background-color: #333; color: white; }
ID selectors should be unique per page. Use class selectors for reusable styles.

Combinators

Descendant Selector (Space)

div p { color: blue; }

Child Selector (>)

ul > li { list-style: none; }

Adjacent Sibling Selector (+)

h1 + p { font-weight: bold; }

General Sibling Selector (~)

h1 ~ p { margin-top: 10px; }

Attribute Selectors

Attribute Presence

[disabled] { opacity: 0.5; }

Attribute Value

input[type="text"] { border: 1px solid #ccc; }

Attribute Starts With

a[href^="https"] { color: green; }

Attribute Ends With

img[src$=".jpg"] { border: 2px solid red; }

Attribute Contains

a[href*="example"] { font-weight: bold; }

Pseudo-classes

Link Pseudo-classes

a:link { color: blue; } /* Unvisited links */ a:visited { color: purple; } /* Visited links */ a:hover { color: red; } /* Mouse over */ a:active { color: orange; } /* Selected link */

Form Pseudo-classes

input:focus { outline: 2px solid blue; } input:checked { transform: scale(1.2); } input:disabled { background: #f0f0f0; } input:required { border: 2px solid red; }

Structural Pseudo-classes

li:first-child { font-weight: bold; } li:last-child { margin-bottom: 0; } li:nth-child(odd) { background: #f0f0f0; } li:nth-child(even) { background: #e0e0e0; } li:nth-child(3) { color: red; }

Pseudo-elements

::before and ::after

.quote::before { content: '"'; font-size: 2em; color: #ccc; } .quote::after { content: '"'; font-size: 2em; color: #ccc; }

::first-line and ::first-letter

p::first-line { font-weight: bold; color: #333; } p::first-letter { font-size: 3em; float: left; margin-right: 10px; }

::selection

::selection { background: #667eea; color: white; }

3. CSS Box Model

Understanding the Box Model

Every HTML element is considered as a box with the following components:

  • Content - The actual content of the box
  • Padding - Clears area around the content
  • Border - A border around the padding and content
  • Margin - Clears area outside the border
Content Area

Box Model Properties

Width and Height

.box { width: 300px; height: 200px; }

Padding

.box { padding-top: 20px; padding-right: 30px; padding-bottom: 20px; padding-left: 30px; /* Shorthand */ padding: 20px 30px; /* top/bottom left/right */ padding: 20px 30px 40px; /* top left/right bottom */ padding: 20px 30px 40px 50px; /* top right bottom left */ }

Border

.box { border-width: 2px; border-style: solid; border-color: #333; /* Shorthand */ border: 2px solid #333; /* Individual sides */ border-top: 3px dashed red; border-right: 3px dotted blue; border-bottom: 3px solid green; border-left: 3px double purple; }

Border Radius

.box { border-radius: 10px; border-radius: 50%; /* Circle */ border-radius: 10px 20px 30px 40px; /* Individual corners */ }

Margin

.box { margin-top: 20px; margin-right: 30px; margin-bottom: 20px; margin-left: 30px; /* Shorthand */ margin: 20px 30px; /* top/bottom left/right */ margin: 20px 30px 40px; /* top left/right bottom */ margin: 20px 30px 40px 50px; /* top right bottom left */ /* Auto margin for centering */ margin: 0 auto; }

Box Sizing

.box-content { box-sizing: content-box; /* Default */ width: 300px; padding: 20px; border: 5px; /* Total width = 300 + 20*2 + 5*2 = 350px */ } .box-border { box-sizing: border-box; width: 300px; padding: 20px; border: 5px; /* Total width = 300px (padding and border included) */ }
Always use box-sizing: border-box for more predictable layouts. It's commonly applied globally with:
* { box-sizing: border-box; }

Display Property

Block Elements

.block { display: block; /* Takes full width, starts on new line */ }

Inline Elements

.inline { display: inline; /* Takes only required width, flows with text */ }

Inline-Block Elements

.inline-block { display: inline-block; /* Flows like inline but can have block properties */ }

None

.hidden { display: none; /* Element is completely hidden and takes no space */ }

Overflow

.overflow-visible { overflow: visible; /* Default */ } .overflow-hidden { overflow: hidden; /* Clips content */ } .overflow-scroll { overflow: scroll; /* Always shows scrollbars */ } .overflow-auto { overflow: auto; /* Shows scrollbars only when needed */ }

5. Responsive Design

What is Responsive Design?

Responsive design is an approach to web design that makes web pages render well on a variety of devices and window or screen sizes. Content, design and performance are necessary across all devices.

Viewport Meta Tag

<meta name="viewport" content="width=device-width, initial-scale=1.0">
Always include the viewport meta tag for responsive design. It tells the browser how to control the page's dimensions and scaling.

Media Queries

Basic Media Query

@media (max-width: 768px) { .container { width: 100%; padding: 10px; } }

Media Query Features

@media (min-width: 768px) and (max-width: 1024px) { /* Styles for tablets */ } @media (orientation: landscape) { /* Landscape styles */ } @media (hover: hover) { /* Devices with hover capability */ } @media (prefers-reduced-motion: reduce) { /* Reduce animations for users who prefer less motion */ }

Common Breakpoints

/* Small devices (phones) */ @media (max-width: 576px) { ... } /* Medium devices (tablets) */ @media (min-width: 577px) and (max-width: 768px) { ... } /* Large devices (desktops) */ @media (min-width: 769px) and (max-width: 992px) { ... } /* Extra large devices */ @media (min-width: 993px) { ... }

Responsive Images

Max-width Technique

img { max-width: 100%; height: auto; }

Picture Element

<picture> <source srcset="image-large.jpg" media="(min-width: 1200px)"> <source srcset="image-medium.jpg" media="(min-width: 768px)"> <img src="image-small.jpg" alt="Responsive image"> </picture>

Srcset Attribute

<img src="image-default.jpg" srcset="image-small.jpg 300w, image-medium.jpg 600w, image-large.jpg 900w" sizes="(max-width: 300px) 280px, (max-width: 600px) 580px, 880px" alt="Responsive image">

Responsive Typography

Viewport Units

h1 { font-size: 5vw; } p { font-size: clamp(14px, 2vw, 18px); }

Fluid Typography

:root { --min-font-size: 16px; --max-font-size: 24px; --min-viewport: 320px; --max-viewport: 1200px; } body { font-size: calc( var(--min-font-size) + (var(--max-font-size) - var(--min-font-size)) * ((100vw - var(--min-viewport)) / (var(--max-viewport) - var(--min-viewport))) ); }

Responsive Layout Patterns

Mobile-First Approach

/* Base styles for mobile */ .container { width: 100%; padding: 10px; } /* Tablet styles */ @media (min-width: 768px) { .container { width: 750px; margin: 0 auto; } } /* Desktop styles */ @media (min-width: 1024px) { .container { width: 960px; } }

Column Drop Pattern

.columns { display: flex; flex-direction: column; } @media (min-width: 768px) { .columns { flex-direction: row; } }

Responsive Navigation

Hamburger Menu

.nav-menu { display: none; } .nav-toggle { display: block; } @media (min-width: 768px) { .nav-menu { display: flex; } .nav-toggle { display: none; } }

6. Animations and Transitions

CSS Transitions

Basic Transition

.box { width: 100px; height: 100px; background: blue; transition: all 0.3s ease; } .box:hover { width: 200px; background: red; }

Transition Properties

.element { transition-property: width, height; transition-duration: 0.5s; transition-timing-function: ease-in-out; transition-delay: 0.2s; /* Shorthand */ transition: width 0.5s ease-in-out 0.2s, height 0.5s ease-in-out 0.2s; }

Timing Functions

.ease-linear { transition-timing-function: linear; } .ease { transition-timing-function: ease; } .ease-in { transition-timing-function: ease-in; } .ease-out { transition-timing-function: ease-out; } .ease-in-out { transition-timing-function: ease-in-out; } .cubic-bezier { transition-timing-function: cubic-bezier(0.68, -0.55, 0.265, 1.55); }

CSS Animations

Keyframes

@keyframes slideIn { 0% { transform: translateX(-100%); opacity: 0; } 100% { transform: translateX(0); opacity: 1; } }

Animation Properties

.animated-box { animation-name: slideIn; animation-duration: 1s; animation-timing-function: ease-out; animation-delay: 0.5s; animation-iteration-count: infinite; animation-direction: alternate; animation-fill-mode: forwards; /* Shorthand */ animation: slideIn 1s ease-out 0.5s infinite alternate forwards; }

Animation Iteration Count

.once { animation-iteration-count: 1; } .twice { animation-iteration-count: 2; } .infinite { animation-iteration-count: infinite; }

Animation Direction

.normal { animation-direction: normal; } .reverse { animation-direction: reverse; } .alternate { animation-direction: alternate; } .alternate-reverse { animation-direction: alternate-reverse; }

Animation Fill Mode

.none { animation-fill-mode: none; } .forwards { animation-fill-mode: forwards; } .backwards { animation-fill-mode: backwards; } .both { animation-fill-mode: both; }

Transform Functions

2D Transforms

.translate { transform: translate(50px, 100px); } .scale { transform: scale(1.5); } .rotate { transform: rotate(45deg); } .skew { transform: skew(20deg, 10deg); } .matrix { transform: matrix(1, 0, 0, 1, 0, 0); }

3D Transforms

.translate3d { transform: translate3d(50px, 100px, 200px); } .scale3d { transform: scale3d(1.5, 1.2, 0.8); } .rotate3d { transform: rotate3d(1, 1, 1, 45deg); } .perspective { transform: perspective(500px) rotateX(45deg); }

Animation Examples

Bounce Animation

@keyframes bounce { 0%, 20%, 50%, 80%, 100% { transform: translateY(0); } 40% { transform: translateY(-30px); } 60% { transform: translateY(-15px); } } .bounce { animation: bounce 2s infinite; }

Fade In Animation

@keyframes fadeIn { from { opacity: 0; } to { opacity: 1; } } .fade-in { animation: fadeIn 1s ease-in; }

Spin Animation

@keyframes spin { from { transform: rotate(0deg); } to { transform: rotate(360deg); } } .spin { animation: spin 2s linear infinite; }

7. Advanced CSS Concepts

CSS Variables (Custom Properties)

Declaring Variables

/* Global variables */ :root { --primary-color: #667eea; --secondary-color: #764ba2; --font-size-base: 16px; --spacing-unit: 8px; } /* Local variables */ .component { --component-bg: #f0f0f0; --component-padding: 20px; }

Using Variables

.button { background-color: var(--primary-color); color: white; padding: calc(var(--spacing-unit) * 2) calc(var(--spacing-unit) * 3); font-size: var(--font-size-base); } .button:hover { background-color: var(--secondary-color); }

Fallback Values

.element { color: var(--custom-color, #333); font-size: var(--custom-size, var(--font-size-base, 16px)); }

CSS Functions

Calc() Function

.container { width: calc(100% - 40px); height: calc(100vh - 100px); margin: calc(var(--spacing-unit) * 2); }

Min(), Max(), Clamp()

.responsive-text { font-size: min(2vw, 18px); width: max(300px, 50%); line-height: clamp(1.2, 1.5vw, 1.8); }

Color Functions

.element { background: rgb(255, 0, 0); background: rgba(255, 0, 0, 0.5); background: hsl(0, 100%, 50%); background: hsla(0, 100%, 50%, 0.5); }

CSS Counters

body { counter-reset: section; } h2 { counter-increment: section; } h2:before { content: "Section " counter(section) ": "; }

CSS Shapes

.circle { shape-outside: circle(50%); float: left; width: 200px; height: 200px; } .polygon { shape-outside: polygon(50% 0%, 100% 50%, 50% 100%, 0% 50%); float: right; }

CSS Blend Modes

.blend-multiply { mix-blend-mode: multiply; } .blend-screen { mix-blend-mode: screen; } .blend-overlay { mix-blend-mode: overlay; } .background-blend { background-blend-mode: multiply; background-image: url('image.jpg'), linear-gradient(45deg, #667eea, #764ba2); }

CSS Filters

.filter-blur { filter: blur(5px); } .filter-brightness { filter: brightness(1.5); } .filter-contrast { filter: contrast(2); } .filter-grayscale { filter: grayscale(100%); } .filter-sepia { filter: sepia(100%); } .filter-multiple { filter: blur(2px) brightness(1.2) contrast(1.1); }

CSS Masking

.mask-image { mask-image: url('mask.png'); mask-mode: alpha; mask-repeat: no-repeat; mask-position: center; } .mask-gradient { mask-image: linear-gradient(45deg, transparent, black); }

CSS Grid Template Areas

.grid-layout { display: grid; grid-template-columns: 200px 1fr 200px; grid-template-rows: auto 1fr auto; grid-template-areas: "header header header" "sidebar main aside" "footer footer footer" ; gap: 20px; } .header { grid-area: header; } .sidebar { grid-area: sidebar; } .main { grid-area: main; } .aside { grid-area: aside; } .footer { grid-area: footer; }

CSS Logical Properties

.logical { margin-inline-start: 20px; /* margin-left in LTR, margin-right in RTL */ margin-inline-end: 20px; /* margin-right in LTR, margin-left in RTL */ margin-block-start: 10px; /* margin-top */ margin-block-end: 10px; /* margin-bottom */ padding-inline: 20px; /* padding-left and padding-right */ padding-block: 10px; /* padding-top and padding-bottom */ border-inline: 2px solid #333; /* border-left and border-right */ border-block: 2px solid #333; /* border-top and border-bottom */ }

CSS Container Queries

.container { container-type: inline-size; } @container (min-width: 400px) { .card { display: flex; flex-direction: row; } } @container (max-width: 399px) { .card { display: flex; flex-direction: column; } }

CSS Architecture Methodologies

BEM (Block Element Modifier)
/* Block */ .card { } /* Element */ .card__title { } .card__content { } /* Modifier */ .card--featured { } .card__title--large { }
SMACSS (Scalable and Modular Architecture for CSS)
/* Base */ body, h1, p { } /* Layout */ .l-header, .l-footer { } /* Module */ .btn, .card { } /* State */ .is-active, .is-hidden { } /* Theme */ .theme-dark { }

.d-flex { display: flex; } .text-center { text-align: center; } .mt-20 { margin-top: 20px; } .p-10 { padding: 10px; }

Performance Optimization

Minimize CSS Size

/* Before */ .button { background-color: #667eea; color: #ffffff; padding-top: 10px; padding-right: 20px; padding-bottom: 10px; padding-left: 20px; } /* After - Optimized */ .button{background:#667eea;color:#fff;padding:10px 20px}

Use Efficient Selectors

/* Inefficient */ div.container div.content ul li a { } /* Efficient */ .nav-link { }

Avoid @import

/* Bad - blocks parallel downloads */ @import url('styles.css'); /* Good - use link tag */ <link rel="stylesheet" href="styles.css">

Maintainability

Use CSS Variables

/* Define variables */ :root { --primary-color: #667eea; --secondary-color: #764ba2; --spacing-sm: 8px; --spacing-md: 16px; --spacing-lg: 24px; } /* Use variables */ .btn-primary { background-color: var(--primary-color); padding: var(--spacing-md) var(--spacing-lg); }

Comment Your Code

/* ============================================ COMPONENT: Button ============================================ */ /* Base button styles */ .btn { display: inline-block; padding: 10px 20px; border: none; border-radius: 4px; cursor: pointer; } /* Primary button variation */ .btn--primary { background-color: #667eea; color: white; }

Follow Naming Conventions

/* Use hyphens for class names */ .main-header { } /* Good */ .mainHeader { } /* Bad */ .main_header { } /* Bad */ /* Be descriptive */ .user-profile-card { } /* Good */ .card { } /* Too generic */ .blue-card { } /* Avoid color names */

Accessibility

Use Relative Units

/* Good - relative units */ body { font-size: 16px; } h1 { font-size: 2rem; /* 32px */ } .container { width: 90%; max-width: 1200px; } /* Bad - absolute units */ h1 { font-size: 32px; /* Not scalable */ }

Provide Sufficient Color Contrast

/* Good contrast ratios */ .text-primary { color: #333333; /* On white background */ } .text-secondary { color: #666666; /* On white background */ } /* Bad - low contrast */ .text-low-contrast { color: #cccccc; /* On white background */ }

Respect User Preferences

/* Reduce motion for users who prefer it */ @media (prefers-reduced-motion: reduce) { * { animation-duration: 0.01ms !important; animation-iteration-count: 1 !important; transition-duration: 0.01ms !important; } } /* Support dark mode */ @media (prefers-color-scheme: dark) { :root { --bg-color: #1a1a1a; --text-color: #ffffff; } }

Browser Compatibility

Use Vendor Prefixes

.element { -webkit-transform: rotate(45deg); -moz-transform: rotate(45deg); -ms-transform: rotate(45deg); -o-transform: rotate(45deg); transform: rotate(45deg); }

Provide Fallbacks

.gradient-bg { background-color: #667eea; /* Fallback */ background: linear-gradient(45deg, #667eea, #764ba2); } .flexible-layout { display: block; /* Fallback */ display: flex; }

Use Feature Queries

@supports (display: grid) { .container { display: grid; grid-template-columns: repeat(3, 1fr); } } @supports not (display: grid) { .container { display: flex; flex-wrap: wrap; } }

Testing and Debugging

Use Browser Developer Tools

  • Elements Panel - Inspect and modify HTML/CSS
  • Computed Styles - See final applied styles
  • Box Model - Visualize padding, border, margin
  • Animation Panel - Debug animations
  • Responsive Design Mode - Test different screen sizes

Validate Your CSS

Performance Testing

  • Use Lighthouse for performance audits
  • Check CSS file sizes and load times
  • Monitor render-blocking CSS
  • Test on real devices and browsers

CSS Units

Absolute Lengths

cm - centimeters mm - millimeters in - inches (1in = 96px = 2.54cm) px - pixels (1px = 1/96th of 1in) pt - points (1pt = 1/72 of 1in) pc - picas (1pc = 12pt)

Relative Lengths

em - relative to font-size of element (2em = 2x font-size) rem - relative to font-size of root element ex - relative to x-height of current font ch - relative to width of "0" (zero) vh - relative to 1% of viewport height vw - relative to 1% of viewport width vmin - relative to 1% of smaller viewport dimension vmax - relative to 1% of larger viewport dimension % - relative to parent element

CSS Functions

Mathematical Functions

calc() - performs calculations min() - returns the minimum value max() - returns the maximum value clamp() - clamps value between min and max

Color Functions

rgb() - red, green, blue rgba() - red, green, blue, alpha hsl() - hue, saturation, lightness hsla() - hue, saturation, lightness, alpha

Transform Functions

translate() - moves element scale() - scales element rotate() - rotates element skew() - skews element matrix() - 2D transformation matrix translate3d() - 3D translation scale3d() - 3D scaling rotate3d() - 3D rotation perspective() - perspective transformation

Gradient Functions

linear-gradient() - linear gradient radial-gradient() - radial gradient conic-gradient() - conic gradient repeating-linear-gradient() - repeating linear gradient repeating-radial-gradient() - repeating radial gradient
