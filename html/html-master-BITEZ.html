What is HTML?

HTML (HyperText Markup Language) is the standard markup language for creating web pages. It describes the structure of a web page using elements and tags. HTML is the foundation of all websites and web applications.

üí° Fun Fact: HTML was created by Tim Berners-Lee in 1991 and has evolved significantly since then. The latest version is HTML5, which introduced many new features and capabilities.
